# qmk-comment-gen
Generates fancy layout comments for qmk keymaps 

## How to use qmk-commgen.py
+ put keymap.c in the same folder as the .py 
+ run qmk-commgen.py 
+ open comment.txt

## What's new?
+ added support for multiple layers
+ Better boxes
+ Clipboard support
