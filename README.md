# qmk-comment-gen
Generates fancy layout comments for qmk keymaps 

## How to use
+ put keymap.c in the same folder as scripts 
+ run qmk-commgen.py 
+ open comment.txt

## What's new?
+ added support for multiple layers
+ now uses keymap.c directly
